Requirements
------------

* Recent TeX installation (tested on a 2017 one)
* XeLaTeX
* [Font Awesome](https://github.com/xdanaux/fontawesome-latex)
* [Fira Font](https://github.com/mozilla/Fira)
* LaTeX packages (available from CTAN, see source)
